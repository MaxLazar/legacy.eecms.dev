Main Configuration

Check member registration

When registering a user, it will compare their data (nickname, IP and mail) with the "Stop Forum Spam" DB. If it match email or Nickname AND IP, then the user will be banned'.

Minimum Frequency

Minimum frequency (number of times that account has been reported by others) before performing any action.

Stop Forum Spam API key

If you would like to be able to submit information to the Stop Forum Spam, sign up for an API key and ener it in this field.

Action

the action to perform when an account is registered and the profile information matches information stored on Stop Forum Spam
