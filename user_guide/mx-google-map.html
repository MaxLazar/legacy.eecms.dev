MX Google Map User Guide¶

MX Google Map fieldtype provides a field for include Google Maps into entries with "store locator" functionality.

  • MSM compatible
  • Based on Google Map API 3
  • Support multi geo fields for entry
  • Support multi markers per entry
  • Drag and drop map marker point for precise control
  • “Store locator” functionality
