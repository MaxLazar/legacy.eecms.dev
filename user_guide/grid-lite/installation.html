Installation

  1. Download the latest version of Grid Lite and extract the .zip to your desktop.
  2. Copy system/expressionengine/third_party/grid_lite to system/expressionengine/third_party/
  3. Copy themes/third_party/grid_lite to themes/third_party/

Activation

  1. Log into your control panel
  2. Browse to Addons >Fieldtypes
  3. Enable all the Grid Lite components
