MX Extended Content Menu User Guide¶

MX Extended Content Menu its small add-on making CP Content menu more useful for users . This Accessory adds a drop-down channel list for Edit submenu. Also, in case if you use Pages, you can get Pages submenu with list of pages in order to access them very quickly.
