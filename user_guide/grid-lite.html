Grid Lite User Guide¶

Grid allows you to dynamically add / delete and sort rows of data similar to a spreadsheet. Each data row can have one or more cells which are assigned to different custom fields (right now is only next fields available: Text input, Textarea, Select box, Checkbox, Dates. More field coming soon!)
